# File Browser

This component combines FileList and FileDisplay together. It also provides the "driving logic" for the sub-components. For example it includes the state for navigating the file system, and determines weather a file should be rendered as markdown or not.

export let exampleDir = {
  fileA: "hello world",
  fileB: "Green eggs and ham",
  dirA: { FileC: "Foo and Bar" },
};

FileBrowser takes a single prop:

- data - which must be a [Directory object](http://localhost:6006/?path=/docs/components-file-browser-outline--page#Directory%20interface)
