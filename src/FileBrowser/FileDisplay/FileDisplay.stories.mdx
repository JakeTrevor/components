# File Display

Component that displays a single file.

The text to be rendered in as the `data` prop.
FileDisplay then does syntax highlighting with [highlight.js](https://highlightjs.org/). Alternatively, if the `isMarkdown` prop is set, it renders the contents as markdown using [react-markdown](https://www.npmjs.com/package/react-markdown).
